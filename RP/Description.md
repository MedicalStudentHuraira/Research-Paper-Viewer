# Research Paper Viewer: A Lightweight, Framework-Free Academic Paper Browser

## Overview
Research Paper Viewer is an open-source, client-side web application designed for researchers, academics, and students to organize, search, and read academic papers in Markdown format. Built with vanilla JavaScript, HTML, and CSS, this lightweight solution requires no external dependencies or complex setup.

## Key Features

### Advanced Search & Discovery
- Full-text search across paper titles, abstracts, and content
- Filter by authors, publication year, and custom tags
- Fast, client-side search with zero server requirements

### Clean Academic Interface
- Distraction-free reading experience
- Responsive design optimized for all devices
- Support for LaTeX mathematical notation
- Syntax highlighting for code blocks

### Performance Optimized
- No external dependencies or heavy frameworks
- Instant loading and rendering
- Minimal resource usage
- Works offline after initial load

### Easy to Extend
- Simple Markdown-based paper format
- JSON configuration for metadata
- Customizable themes and layouts
- Open-source MIT licensed

## Use Cases
- Personal research paper organization
- Academic literature review
- Research group knowledge base
- Educational resource repository
- Conference paper management

## Technology Stack
- **Frontend**: Vanilla JavaScript (ES6+)
- **Styling**: Pure CSS with CSS Variables
- **Markdown Parser**: Custom lightweight implementation
- **Search**: Client-side search algorithm
- **Deployment**: Single HTML file deployment

## Why Choose Research Paper Viewer?
- **Privacy First**: Your papers stay on your machine
- **No Vendor Lock-in**: Open formats (Markdown/JSON)
- **Future-Proof**: No framework dependencies
- **Accessible**: Works on any modern browser
- **Lightweight**: Under 50KB of JavaScript

## Getting Started
1. Clone the repository
2. Add your papers in Markdown format
3. Update the papers.json index
4. Open index.html in any browser
